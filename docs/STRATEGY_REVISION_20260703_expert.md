## 0. 一句话结论

**本文的科学价值与审稿防御力都在 Stage C（中子学保真度），不在 Stage A（PIC）。**
把"高精度 3D PIC"当可信度核心，是把资源投在了最贵、且最容易被等离子体审稿人攻击的环节。
可信度的正确来源是：**① 验证过的中子学管线 + ② 结论对源的稳健性（参数化畸变扫描把 PIC 点括在中间）**。
PIC 只提供"一个物理上合理的锚点"，不是承重墙。

---

## 1. 为什么推翻"3D 优先 / 3D 最准确"

1. **任何可负担的固体密度 3D 都欠解析趋肤深度。** n_e≈183 nc 时 c/ω_pe≈9.4 nm；任何能在单节点/合理机时内跑完的 3D，横向网格都在 40–65 nm 量级，欠解析约 4–6×。花几百元跑一发仍不收敛的 3D，等离子体审稿人一眼识破——等于**主张了守不住的精度**，反而落把柄。
2. **承载结论的物理不依赖 PIC 维度。** 下游 OpenMC 输运 + ENDF/FENDL 截面 + ⁷Li 阈值反应，都是核工程验证过、审稿人信任的组件。真正承重的是"源畸变 → ⁷Li 超额 TPR"这个**映射**，它只依赖源的两个标量特征：**>2.82 MeV 中子占比**与**前向程度**。这两个数由 2D、3D 还是参数化模型给出，下游一致。
3. **成本/风险/边际价值三重不划算。** 继承记录显示 3D 反复 OOM、超时、parser 失败，已消耗可观机时而无一发有效源；而它能为论文增加的东西非常有限。

## 2. 可信度架构（防质疑的正确立法）

- **腿一 · 主干（承重）= 验证过的中子学管线 + 参数化畸变稳健性扫描。**
  扫 >2.82 MeV 占比与前向张角，画"畸变强度 → ⁷Li 超额 TPR"曲线，证明结论在整个真实畸变区间内都成立、且随 >2.82 MeV 占比按一条曲线变。结论不依赖任何单一源的偶然谱形。
- **腿二 · 锚点（点缀，便宜）= 已有 2D 源 + 一发便宜 3D。**
  仅用于证明"真实源落在参数化畸变带内、2D→3D 的谱-角偏移有界"。**不扫 3D、不追收敛。**

### 3.3 "so what"（决定能否发好，务必写进 Abstract/Discussion）
落点**不是**"硬谱产更多阈值中子"（trivial，会被怼）。落点是给用紧凑/激光 D-D 源做增殖包层试验台的社区（Astral、UKAEA LIBRTI、CFS 包层）一个可操作修正：

> **若把真实（畸变）源当作理想 2.45 MeV 各向同性源来解读产氚测量，会在 ⁷Li 阈值窗口（>2.82 MeV）和前向角区系统性偏差 X%——本文给出该修正量及其随 >2.82 MeV 占比的标度。**

## 4. 必须补的三个硬洞（不补易被毙）

1. **D(d,n) 的 CM 各向异性**（接口已留 `cm_dir_unit`）。论文主题即"各向异性"，却对核反应本身用各向同性，是最易一击致命的洞。用 Legendre/ENDF 参数化实现，保留各向同性作可切换基线，出"带异性 vs 各向同性"敏感性图。**勿破坏现有 `gate_kinematics`。**
2. **真实阻止本领**：用 SRIM/PSTAR 导出数据替换现 `data/stopping_D_in_CD2.csv` 的 provisional seed，决定绝对产额可信度。
3. **多库不确定度**：⁷Li 关键截面 ENDF/B vs FENDL-3.2 对比一版，把不确定度写在明处。

## 5. PIC 源定义修订（弃 rear+20µm/5ps）

**弃用**继承的"rear+20µm 源面 + 积分到 5 ps + 末窗<10% 完整性门槛"。该要求是"源面取在靶后 20µm、逼慢氘在 PIC 里飞过整段真空"的人为产物；氘离开靶背鞘层后是无场自由飞行，这段漂移应由 Stage B 用运动学补，PIC 不必陪跑。

**新定义**：PIC 只需跑到 **TNSA 加速定型**（靶背鞘层场衰减）。
- 探针面贴近靶：`rear + 3~5 µm`（刚出鞘层）。
- `t_end ≈ 1 ps`（用 2–3 个快照验证 **E_max 与 >2.82 MeV 产额占比已 plateau** 即停）。
- **D PPC 拉高**（如 `e/D/C = 8/32/4` 或更高）：论文信号在高能氘的稀疏尾，尾巴统计质量由 D PPC 直接决定；电子够解析鞘层、碳撑住电荷/场即可。
- 物理依据：产 >2.82 MeV 中子的高能氘最早、最快形成，几百 fs 即抓全；被 5 ps 苦追的低能宽角尾主要喂 ⁶Li 主通道，而 ⁶Li 对源细节钝感。

**便宜 3D 锚点网格（C 变体，单节点 256 核估几十元）**：
`nx,ny,nz = 1500,320,320`，`x=[-8,30]µm`，`y,z=[-10,10]µm`，靶 3 µm/±5 µm，
强制拓扑 `nprocx,nprocy,nprocz = 16,4,4`，`restart_dump_every=-1`、`force_final_to_be_restartable=T`，`particles=never`。
先跑 300 fs 微基准实测 `s/step` 与 `MaxRSS`，再定正式那一发的 t_end 与核数。

## 6. 不要过度主张（守边界）

- 不写"激光能产氚"（能量账差几个数量级，严谨做下去得反面）。
- 不写"激光是更好的中子/产氚源"（稳态高通量/成本加速器全面占优）。
- 激光/紧凑源是**背景与动机**（IFE 电站中子本就畸变脉冲、无"换加速器"选项），不是论点。
- 只聚焦 D-D 分支；CD2 中 `12C(d,n)`、体相反应、hole-boring 等作为假设边界显式声明。
