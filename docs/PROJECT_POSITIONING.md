# 项目定位与研究框架 (Project Positioning)

> 本文件解决"这个项目到底在证明什么、为什么可信、下一步往哪走"。
> 结论: **不做"证明激光可产氚",改做"评估紧凑/激光 D-D 源作为产氚试验台的源保真度"。**

## 0. 2026-07-03 执行口径

用户已明确选择 **3D PIC 作为主物理锚点**，理由是三维源的谱-角-横向发散才更接近真实实验。本文档中关于 2D/参数化源的论证不再表示"用 2D 替代 3D"，而表示:

- 3D PIC 给真实源锚点，优先做 `a0=10` 单点端到端链路；
- 参数化源扫描给稳健性和审稿防御力，证明结论不是某一次 PIC 源的偶然结果；
- 已完成的 2D 源只用于代码开发、历史对照和参数区间参考，不作为最终真实性核心。

当前 3D 候选模板已收敛到 `hpc/templates/epoch3d_stage1_benchmark_3um_512_smoke.deck` 与 `hpc/templates/epoch3d_stage1_benchmark_3um_512_full.deck`。第一步是 **50 fs smoke**，确认解析/MPI/内存/输出；通过后再做 **1500 fs 四探针 benchmark**，用实测 wall-clock、内存、scratch 和探针结果决定正式 3D 作业的 walltime/核数/成本。本文档中的 3D 成本数字均为提交前估算或历史保守估算，不能当作报价。

## 1. 背景动机(为什么这个方向是真的)

紧凑 D-D 聚变中子源 → 锂增殖包层产氚,是 2025 年真实且活跃的产业方向:

- **Astral Systems**(英,私营): 首个用自家 D-D 聚变中子源照射锂增殖包层、并实测到氚的商业公司(55 小时 D-D 照射)。这几乎就是本项目概念本身。
- **UKAEA LIBRTI**(£220M): 专门"演示净产氚"的计划; **CFS** 已加入包层测试。
- **SHINE Technologies**: 供 UKAEA 14 MeV D-T 中子源(世界最亮稳态源,~5e13 反应/秒)。
- **LLNL LD-FIRST**: 激光驱动聚变集成研究白皮书。

注意: 主流紧凑源多为**加速器/静电型**,不是激光; "激光驱动 D-D 专门产氚"是更小众角落。定位时要吃准这一点。

## 2. 诚实的定位: 评估"保真度/生态位",不是"证明可行"

**能量-经济粗账(务必在论文里给一版正式的):**
激光→氘 效率 ~1-10%,氘→中子 ~1e-5,单发大激光 ~1e8-1e10 中子。
产 1 g 氚(~2e23 原子,乐观取 TBR≈1)需 ~2e23 中子 → **~PJ 级激光能量/克**,折合几百 GWh 电/克;
氚市价 ~$3 万/克。作为**主力**产氚手段差好几个数量级,不成立。

因此本项目**不设为"证明激光可产氚"**(严谨做下去会得到反面),而设为:

- **(A) 源保真度问题**: 用非理想(畸变)D-D 源做增殖包层试验台/mock-up 时,
  相对理想源,产氚测量有无系统性偏差、偏在哪个通道/能段。**这正好用上现有的"畸变源 vs 理想源、⁷Li 通道"研究。**
- **(B) 生态位评估**: 激光源独有的脉冲、前向定向、高瞬时通量能做什么常规源做不了的事,
  而不是把它当主力产氚炉。

## 2.5 为什么用激光源(而非证明其"更优")

**不要在"激光比加速器好"上立论——会输。** 稳态高通量/成本/可复现/可标定,加速器全面占优
(SHINE 稳态 D-T ~5e13/s; Astral 用非激光; 传统 D-D 发生器又便宜又稳)。切勿写"激光是更好的产氚/中子源"。

**正确立论: 激光是背景/动机,不是论点。**
- 在激光/惯性聚变(IFE)电站里,中子本就来自激光驱动聚变——脉冲、畸变、**没有"换加速器"这个选项**
  (Focused Energy 激光 D-T IFE、LLNL IFE)。增殖包层将来面对的就是这种畸变脉冲场。
- 因此"凭什么用激光"**自答**: 因为 IFE 电站的中子就从那儿来。本文研究的是"在这种源环境下产氚的保真度",
  受众是已在搞 IFE 的人,不是去说服别人弃用加速器。

**激光独有、可诚实主张的次要卖点(勿夸大):**
- 超短脉冲 / 极高瞬时通量 → 时间分辨、瞬态辐照研究。
- 宽能谱多 MeV 尾 → 天然产出 >3.1454 MeV 中子,正好碰当前 OpenMC `Li7` MT205 `(n,Xt)` 产氚阈值; 近单能 2.45 MeV D-D 发生器碰不到。
- µm 源点 → 利于飞行时间/成像。

**正面接住"束-靶=更差加速器"的质疑:** 本设计(激光→氘束→外置 CD2 converter；TiD2 作为未来材料扩展)作为源技术确实类加速器、
稳态不占优; 但价值在 (a) 代表 IFE 包层真实会看到的畸变脉冲场, (b) 脉冲/宽谱/高瞬时是加速器给不了的
**另一类**中子场。把限制写在明处,反而显得想清楚了。

## 3. 第一篇小论文(本月)重新定题

- **旧题(弃)**: 证明激光驱动产氚可行 —— 会被能量账打死。
- **新题**: 面向紧凑 D-D 源产氚试验台,评估**源谱-角畸变对锂靶产氚的保真度影响**。
  主结果不变: 畸变的影响集中在 ⁷Li 产氚阈值窗口(当前 OpenMC MT205 为 >3.1454 MeV),⁶Li(n,α)T 钝感。

**为什么这个主张在 3D 锚点 + 参数化源下可信(核心论证,写进方法/讨论):**

1. **PIC 源不是研究对象,是畸变参数空间里的一个"物理锚点"。** 当前执行上这个锚点优先采用 3D。
   真正的自变量是"畸变有多少"(>3.1454 MeV 占比、前向程度)。结论稳健性来自**3D 锚点 + 参数化畸变扫描**共同支撑。
2. **2D 若被使用,只作为对照或保守边界。** 2D 往往高估前向聚焦与离子截止 = 把畸变夸大。可写:
   "即便用 2D 夸大过的畸变,总 TPR 影响也仅 X%"。
3. **承载结论的物理是标定过的**: ⁷Li 阈值反应 + ENDF/FENDL 截面 + OpenMC 输运,
   均为核工程验证过的组件; GATE 体系在做分件验证(Bosch-Hale 对 ENDF、运动学对解析、OpenMC 对基准)。
   端到端不对真实打靶验证,但这类耦合模拟在此阶段皆如此——靠"分件验证 + 透明声明理想化"取信。
4. **主张是条件性、相对、机理层**的("若源畸变如此,则 TPR 影响如此"),不是绝对预测。
   绝对 per-shot 产氚数仅作 illustrative(2D 归一)。

## 4. 据此的方法学调整

1. **参数化源从"备份"升为"主干"**: 扫 `kT ∈ {1,2,3} MeV`、`theta_max ∈ {10,20} deg`,
   换算成 >3.1454 MeV 中子占比,画**"畸变强度 → ⁷Li 超额 TPR"曲线**; PIC 点叠加当锚。
2. **完整度判据从"氘权重"改为"产额/TPR 加权"**(详见 AGENT_HANDOFF): 源在 ~3.5ps 即达标而非 5ps,
   省机时、更可辩护,并能避免 L_pre=1 那类超时。
3. **报告方式**: 当前论文主结果收窄为 per-source-neutron、Li6/Li7 分道、天然(7.59%) vs 富集(90%);
   可加一个**近纯 Li-7 诊断案例**把畸变信号最大化(不现实但干净,做展示图)。
4. **纳入 CM 各向异性**(详见 AGENT_HANDOFF 建议 2)。

## 4.5 2D→3D 偏差与可信度策略

**可信度靠两条腿:**
- **腿一(主锚点): 1 发做对的 3D,给全分布。** 提供完整 dN/dE + dN/dθ(不是标度律的单个截止能),
  用来 (i) 量化 2D 谱/角偏差, (ii) 确认 3D 落在参数化区间内, (iii) 给 2D→3D 定全谱全角偏差带。
- **腿二(稳健性): 让结论对源稳健。** 参数化扫描把畸变参数(>3.1454 MeV 占比、前向张角)
  扫成一个明显括住 3D 的区间。证明整个区间内效应都落在 ⁷Li 窗口、幅度随 >3.1454 MeV 占比按一条曲线变。
  这样结论不依赖任何单源的偶然谱形。

**注意: 2D↔3D 标度律只覆盖最大/截止能,pin 不住完整谱形 + 角分布 + 绝对产额**,故仅作截止能一项的旁证,
不能当主要可信度依据。

**3D 校验怎么做(1 发、做对、别粗化、别全 3D 扫描):** 分辨率守 ≤10-15nm(固体密度趋肤深度 ~10nm);
单 a0(如 a0=10); 盒子与时间砍到刚够拿源(靶附近 ~1-1.5ps,下游 ~30µm),不追 5ps 完整源、只取 2D→3D 比值。

**单发 3D 机时预算(历史保守粗估,以现有 2D 例标定; 正式提交前先跑微基准校准):**

下表保留为风险上界提醒，不代表当前 resource-controlled 3D deck 的正式价格。当前执行以 50 fs smoke 和后续 1500 fs benchmark 实测为准。

| 方案 | dx | 盒子/时间 | ~core-hours | ~CNY@0.1 | RAM | 节点(768GB) |
|---|---|---|---|---|---|---|
| A 激进最小 | 15nm | 40×30×30µm, 1.0ps | ~2.8e5 | ~2.8万 | ~2.3TB | ~3 |
| B 稳妥 | 12.5nm | 45×48×48µm, 1.5ps | ~8.8e5 | ~8.8万 | ~6.8TB | ~9 |
