# 一个月小论文执行框架
## 激光驱动 D-D 脉冲中子源驱动外部锂靶产氚特性的蒙特卡罗研究
### 技术路线：PIC(EPOCH) → 半解析源项(Python) → 中子学输运(OpenMC)

> **本框架的核心决策：放弃 Geant4，用 Python 半解析生成器替代 Stage 2。**
> OpenMC 不能做氘束-靶聚变（它不输运初级带电粒子）。因此 Stage 2 不是"换成 OpenMC"，
> 而是"换成一段自写的解析中子源项代码"。这样全链路都在 Python/HPC 舒适区内，是一个月可完成的前提。

## 0. 一个月唯一要回答的科学问题（务必收窄）

> **一句话结论型问题：** 激光束-靶产生的"畸变谱"（多普勒展宽 + 前向各向异性）中子，
> 打进锂靶后的空间产氚率 TPR，与理想的各向同性单能 2.45 MeV 源相比差多少、差在哪？

**预期答案（也是卖点）：** 差异不是均匀的，而是**集中在 ⁷Li(n,n'α)T 的阈值窗口（~2.8 MeV）附近，并且随角度变化**。
- ⁶Li(n,α)T 主通道：靠慢化后的热/超热中子，对源谱细节**钝感** → 两种源结果接近。
- ⁷Li(n,n't) 阈值通道：Q≈−2.47 MeV，实验室阈值 ≈ 2.8 MeV。热核 D-D 中子 2.45 MeV 本来卡在阈值下方；
  只有**束-靶前向被顶过 2.8 MeV 的高能尾巴**才能开启这条道。所以这条通道对"谱里有多少料在 2.8 MeV 以上"和"往哪个方向"极度敏感。

**这就是本文的物理新意**：把 Stage 1/2 的"源畸变"和 Stage 3 的"⁷Li 阈值敏感通道"缝在一起，给出一条定量映射。
不要把论文写成"我耦合了两个码"，要写成"我发现畸变源对 TPR 的影响集中在 ⁷Li 阈值窗口且各向异性可观测"。

## 1. 全链路数据流（三个模块 + 两个接口文件）

```
[模块A] EPOCH PIC                 [模块B] Python 半解析源项            [模块C] OpenMC 输运
激光打 3–5 μm CD2  ──> deuteron_beam.h5 ──> 厚靶 D-D 产额+运动学 ──> neutron_source.h5 ──> 锂靶 TPR 比对
(拉出氘束相空间)      (E,方向,权重)         (能谱×角分布×时间)      (E,Ω,t 分布)      (Li-6/Li-7 分道 + 多库)
```

两个接口文件是全项目的"契约"，**先把这两个文件的格式定死，三个模块就能并行开发**：

- `deuteron_beam.h5`：字段 `E`(MeV), `dir`(单位矢量 nx,ny,nz), `weight`(宏粒子权重/真实氘数), 可选 `t`(fs)
- `neutron_source.h5`：字段 `E`(MeV), `dir`, `weight`(中子数/shot), `t`(ns), 附标量 `Y_total`(每 shot 总中子数)

## 2. 模块 A —— PIC 氘束（EPOCH）

**目标产物：** 一个真实的氘束相空间 `deuteron_beam.h5`。

**做法：**
1. EPOCH 2D，靶：厚 3–5 μm 的固体密度 CD₂（`a0`=5~30 扫，脉宽/能量固定一档即可，先跑 1 档跑通）。
2. PPC ≥ 100，分辨率保证能看到 TNSA 鞘层场把靶后氘拔出。
3. 在靶后 x≈20 μm 设虚拟采样面；在激光作用结束、氘自由飞散时（约 0.8–1 ps）导出穿过该面的氘的 `E, 方向, 权重`。
4. Python(SDF reader) 存成 `deuteron_beam.h5`。

**⚠️ 红线：** 靶厚 ≤ 5 μm，绝不把 1 mm 塞进 PIC，否则内存爆炸。

**⏱️ 降级预案（若 EPOCH 排队/太慢，第 1 周就启用）：**
> 用**参数化 TNSA 氘谱**顶替，先把 B/C 全跑通：
> 指数谱 `dN/dE ∝ exp(−E/kT)`，`kT` 取 1–3 MeV；角分布用半角 10°–20° 的锥形前向束。
> 论文里明确写"氘源采用 PIC / 参数化两种输入做对比"，真实 PIC 数据后补即可。
> **这一步是保一个月不翻车的关键——下游绝不能等 PIC。**

## 3. 模块 B —— 半解析厚靶 D-D 中子源项（纯 Python，本项目的技术核心）

**这是替代 Geant4 的部分。物理清晰、可验证、agent 最好写。**

### 3.1 厚靶中子产额（决定绝对归一化）

一个初能 E₀ 的氘在 CD₂ 里慢化，产额（每入射氘的中子数）：

```
Y = ∫_0^{E0}  n_D · σ_DD(E) / S(E)  dE
```
- `n_D`：CD₂ 中**靶氘**的数密度 [cm⁻³]（由 CD₂ 密度 ~1.06 g/cm³ 算出，每分子 2 个 D）
- `σ_DD(E)`：D(d,n)³He 聚变截面，用 **Bosch-Hale 参数化**（S-factor 形式，keV–MeV 段标准公式，agent 可直接实现）
- `S(E) = −dE/dx`：氘在 CD₂ 的阻止本领 [MeV/cm]
  - 数据源三选一：**SRIM 导出表** / **PSTAR(质子)按同速度换算到氘** / Andersen–Ziegler 解析式
  - 换算关系：同速度下 `S_d(E_d) ≈ S_p(E_d/2)`（氘、质子电荷相同、速度相同则电子阻止近似相等）

**校验点①**：单能氘（如 1 MeV）算出的 Y 要和文献厚靶 D-D 产额量级一致（~10⁻⁵ 量级/氘，具体查表核对）。**对不上先别往下走。**

### 3.2 中子能量-角度（决定畸变谱形状 = 论文卖点）

对每个氘、在每个慢化能量 E 处，D(d,n)³He 出射中子：
- CM 系近似**各向同性**（低能段成立；>1 MeV 有轻微各向异性，可先忽略，讨论里提一句）
- 用**两体运动学**把 CM 中子能量/方向 boost 到实验室系：入射氘动量方向 = 该氘的飞行方向（来自模块 A）
- 结果：**前向中子被顶高（可 >2.8 MeV），后向被压低（<2.45 MeV）** —— 这就是各向异性 + 谱展宽的物理来源

Q 值：`D(d,n)³He, Q = +3.27 MeV`。两体运动学出 `E_n(θ_lab, E_d)` 的闭式解，agent 可写。

### 3.3 采样与输出

```
对 deuteron_beam.h5 里每个氘宏粒子:
    沿其慢化路径按 σ_DD(E)/S(E) 采样"发生聚变的能量 E"
    在 CM 各向同性采样中子发射方向
    用氘的飞行方向做 boost → 得 (E_n, dir_n)
    时间 t：可先设 delta 脉冲，或按氘到达时间铺一个 ps–ns 展宽
    累加到 (E_n, Ω, t) 直方图，权重 = 氘权重 × Y_微分
归一化到 每 shot 中子数 Y_total
存 neutron_source.h5
```

**校验点②**：画出中子**能谱**（应见 2.45 MeV 峰 + 被展宽/前向拉出的高能尾）和**角分布**（应前向偏置）。
形状不对（比如没有前向偏置、没有高能尾）说明 boost 写错了。

> **本模块直接对应你第一篇小论文的主结果图：能谱展宽 + 角分布各向异性。**

## 4. 模块 C —— OpenMC 锂靶输运与 TPR

### 4.1 几何与材料
- 锂靶：金属 Li 圆柱（先做 Ø20 cm × L20 cm）或球壳，中心留小腔放源。
- 两套材料各跑一遍：**天然 Li**（⁶Li 7.5%）与 **⁶Li 富集**（如 90%），看富集对结论的影响。
- 源置于中心：用 `openmc.IndependentSource`，能量/角度从 `neutron_source.h5` 的分布构造
  （能量用 `openmc.stats.Tabular`，角度用自定义 `AngleDistribution` 或按方向分 bin 建多个源加权）。

### 4.2 Tally（**必须把 ⁶Li 和 ⁷Li 分开**）
- 产氚率按**核素分道**：对 ⁶Li 和 ⁷Li 分别建 nuclide filter，score 用氚产生反应
  （score 关键字与 MT 号请**对照当前 OpenMC 文档核实**：⁶Li(n,t)⁴He 通常是 MT=105；⁷Li(n,n't) 是含 triton 的非弹道；
  实操上可用 `H3-production` 类总氚产额 + nuclide filter 拆两核素，或直接指定 MT 列表。**别凭记忆写 score 字符串，查文档。**）
- **空间分布**：加 `openmc.RegularMesh`（如轴向 + 径向）做 mesh tally，得到 TPR(r,z) 剖面。
- 能量分辨：给 tally 加 energy filter，看 ⁷Li 通道贡献是否集中在 >2.8 MeV。

### 4.3 两个对照 case（论文的对比核心）
| Case | 源 | 用途 |
|---|---|---|
| A（理想） | 各向同性 + 单能 2.45 MeV 点源 | 传统核工程"完美源"基准 |
| B（真实） | 模块 B 的畸变源（展宽+各向异性） | 激光源 |

**输出对比：** 总 TPR、⁶Li vs ⁷Li 分道占比、TPR(r,z) 空间剖面、A/B 相对偏差(%)及其空间/能量分布。

**校验点③**：TBR/TPR 数量级要物理合理；⁶Li 通道 A≈B（钝感），⁷Li 通道 A、B 差异明显（敏感）。
若两通道都无差异，检查源是否真的畸变了（回模块 B）。

### 4.4 多库比对（时间够就做，不够放讨论/展望）
挂 `FENDL-3.2` / `ENDF/B-VIII.0`(或 VIII.1) / `JEFF-3.3` 各跑一遍，给 TPR 的"误差带"。
> 这块加分但**不是一个月的必需项**。若时间紧，先只用 FENDL-3.2 出主结论，多库作为"下一步"。

## 6. 论文骨架（小论文）

**拟定题目**：《激光驱动厚 CD₂ 靶脉冲 D-D 中子源的谱-角畸变及其对外部锂靶空间产氚率的影响》

1. 引言：氚自持痛点 → 传统 14 MeV/稳态源的局限 → 紧凑激光源的机会与"畸变谱"疑问
2. 方法：PIC→半解析源项→OpenMC 三段耦合（强调 Stage 2 半解析、说明为何不用 Geant4：低能聚变截面稳健性）
3. 结果一：激光 D-D 源的能谱展宽与前向各向异性（模块 B 图）
4. 结果二：锂靶 TPR，A(理想) vs B(畸变) 的总量、⁶Li/⁷Li 分道、空间剖面
5. 结果三（核心洞见）：偏差集中于 ⁷Li 阈值窗口(>2.8 MeV)且各向异性，⁶Li 通道钝感
6. （可选）多库不确定度带
